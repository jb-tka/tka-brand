# Colors

Official school colors (per tka.net): **Red, White, and Royal Blue.** Mascot: Lion.

| Color | Hex | Source |
|---|---|---|
| Royal Blue | `#003580` | live tka.net theme CSS (primary brand blue) |
| Red        | `#D21034` | live tka.net theme CSS (primary brand red) |
| White      | `#FFFFFF` | — |

**Use these as the standard digital palette** — verified 2026-06-16 as the dominant colors in the live tka.net theme CSS (92× blue / 75× red, far ahead of any other brand color).

Rasterized logo PNGs vary slightly from the palette and from each other (the POD logo samples ~blue `#193768` / red `#EA1C2D`; the primary wordmark ~`#005898` / `#E01830`). When compositing a logo itself, match that logo's own art; for text, headers, rule lines, and shapes, use the official palette above.

> The official **brand guide** remains the final authority for print (CMYK/Pantone) and any secondary/neutral palette.
