# Typography

The TKA website loads two typefaces from Google Fonts:

- **EB Garamond** (serif) — weights 400/600, regular + italic
- **Open Sans** (sans-serif) — weights 300/400/600/700, regular + italic

Likely split (confirm against the brand guide): **EB Garamond for headings/display**, **Open Sans for body**.

Web import:
```html
<link href="https://fonts.googleapis.com/css2?family=EB+Garamond:ital,wght@0,400;0,600;1,400;1,600&family=Open+Sans:ital,wght@0,300;0,400;0,600;0,700;1,300;1,400;1,600;1,700&display=swap" rel="stylesheet">
```

> Source: live tka.net CSS. The athletics wordmarks use a separate display face (`Athletic-Regular.ttf`).
